# CMS Report 프로젝트 변경 사항

## 개요
이 문서는 CMS Report 프로젝트의 변경 사항을 기록합니다.

## 변경 사항

### 1. 색상 변경
- **주요 색상 변경**: 모든 텍스트 컬러 `#3E7EFF`를 `#58C65D`로 변경
- **SVG 컬러 변경**: SVG의 `#92B7FF` 컬러를 `#B5FFB8`로 변경
- **인사이트 프레임 배경색**: `#E9F1FF`를 `#EEF9EF`로 변경
- **인사이트 프레임 stroke**: `#9BD39D`로 변경

### 2. 목차 페이지 스타일
- **스케줄명 텍스트 컬러**: 목차 페이지의 `{{스케줄명}}` 텍스트 컬러를 '목차' 텍스트와 동일하게 설정
- **Breadcrumb 그룹 크기**: 목차 페이지의 `breadcrumb-group` 크기를 다른 페이지와 동일하게 조정

### 3. 페이지 13 레이아웃 변경
- **구조 변경**: 피그마 디자인에 맞춰 페이지 13의 레이아웃 재구성
  - 운영 사이트 이미지 제거
  - 미디어그룹 지도와 미디어그룹 광고 개요를 하나의 컨테이너(`frame-media-map-overview-container`)로 그룹화
  - 미디어그룹 광고 개요 테이블을 지도 영역 우측에 배치
- **정렬**: `frame-media-map-overview-container`를 페이지 좌측 중앙으로 정렬
- **크기 조정**:
  - 지도 영역: 너비 400px, 높이 400px
  - 미디어그룹 광고 개요 영역: 높이 400px
  - 미디어그룹 광고 개요 테이블: 높이 hug (auto)
  - 미디어 대수 필드: 높이 fill (100%)
  - 미디어 대수 필드 상하 여백: 12px

### 4. 미디어그룹 광고 개요 테이블
- **테이블 구조**: 피그마 디자인에 맞춰 각 필드를 개별 행으로 재구성
- **미디어 대수 필드**: 
  - 높이를 fill로 설정하여 부모 행의 높이를 채우도록 함
  - 상하 여백 12px 설정
  - 텍스트 수평 중앙 정렬

### 5. 필드 스타일 조정
- **`.field-tall` 필드**: 
  - 높이 및 여백 원상복구
  - 페이지 13/14에서 높이 48px, 패딩 0px 16px로 설정
  - 미디어 대수 필드의 라벨과 값 필드 높이를 fill(100%)로 설정

### 6. 유사 미디어그룹 관련 내용 삭제
- HTML에서 유사 미디어그룹 관련 인사이트 박스 2개 삭제
- HTML에서 유사 미디어그룹 헤더와 행들 삭제
- JavaScript 코드에서 `유사미디어그룹` 필드 및 색상 처리 로직 제거
- CSS에서 유사 미디어그룹 관련 스타일 모두 삭제:
  - `.frame-insight-notice` (유사 미디어그룹 전용)
  - `.similar-media-group-header` 관련 스타일
  - `.similar-media-group-row` 관련 스타일
  - `.field-tall-similar` 스타일
  - `.field-value-item-similar` 스타일
  - `.field-text-small-similar` 스타일

### 7. 파일 제목 변경
- HTML 파일의 `<title>` 태그를 "cms-report"로 변경

### 8. Copyright 연도 자동 입력
- **기능**: 마지막 표지 페이지의 copyright 연도가 보고서 발행 연도로 자동 입력됩니다
- **구현 방식**:
  - HTML에서 copyright 연도 부분에 템플릿 변수 `{{보고서발행연도}}`를 사용
  - 템플릿 엔진에서 보고서 발행 연도를 자동으로 치환합니다
  - 다른 템플릿 변수(`{{보고서발행일}}`, `{{스케줄명}}` 등)와 동일한 방식으로 처리됩니다

## 기술 스택
- HTML5
- CSS3
- Chart.js (차트 라이브러리)
- Pretendard 폰트

- `adex-report.html`: 메인 HTML 파일
- `styles.css`: 스타일시트 파일
- `TEMPLATE_VARIABLES.md`: 템플릿 변수 문서
